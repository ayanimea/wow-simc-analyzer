# wow-simc-analyzer
WoW SimC Analyzer aims to extract a list of the best stuff to farm for a given WoW character.


## Easy execution

### Environment preparation

```sh
pip3 install pytest requests beautifulsoup4 openpyxl
```
### Execution

#### Program
```sh
python3 prototype.py
```

#### Test
```sh
python3 -m pytest
```
